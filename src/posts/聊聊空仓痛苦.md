# 0x01 空仓的客观必要性

### 1.1 为什么要空仓

之前分析过，为什么要在绝大部分时间保持空仓/轻仓：

- 游戏规则：市场会奖励活下来的人，严惩经常梭哈的人
- 规则原因：哪怕是绝佳的操作机会，拉长时间线来看，也总是存在的



### 1.2 如何做到空仓

之前分析过，如何做到空仓：

- 庙算是否易胜：通过分析市场看清自己的手牌，如果是27杂色，那就放聪明点乖乖丢掉

- 27陷阱：谋小利而断大谋。我愿意为了眼前27杂色，放弃来日拿到AA/KK手牌的机会吗？



### 1.3 时常缺席的理智

对之前的分析进行梳理，发现存在局限性：

- **过于客观**：为什么要习惯轻仓、庙算是否易胜，这两个都是客观的游戏规则和游戏策略，不代表人就能遵守

- **过于表面**：27陷阱看似有效的一问，实际上还是很理智，人有时候拿到27还是会忍不住梭哈



现象/问题：时常失效的理智

- 为什么散户在下行的市场，仍然时常忍不住全仓？

- 为什么散户拿到27杂色也会忍不住梭哈？







# 0x02 痛苦驱动：为了逃离痛苦而交易

### 2.1 人体情绪的优先性

人类的生理机制：

- 情绪的优先性：情绪是更原始、更快速的反应系统（比如恐惧、愤怒、渴望），往往先于理智启动。当情绪强度足够高时，会直接劫持决策过程
  - 比如明知熬夜伤身体，但焦虑情绪下仍忍不住刷手机；
  - 明知冲动消费会超支，但当下的愉悦感让人难以克制。

- 理智的滞后性：理智需要时间处理信息、权衡利弊，但情绪反应是即时的。
  - 比如吵架时明知冷静沟通更有效，但愤怒情绪已经触发了脱口而出的指责，事后才后悔。

- 认知资源的有限性：当人处于情绪波动（如压力、疲惫、兴奋）中时，用于理性思考的认知资源会被占用，导致理智难以充分发挥作用。
  - 比如考试前明明知道该复习，却因过度紧张而无法集中注意力，反而拖延。





### 2.2 二七梭哈现象

股市现象：散户明知市场不好，但还是会抱着幻想重仓买入。

德州现象：玩家明知27杂色是败率最高的手牌，还是抱着幻想梭哈。



贪婪与逃离痛苦的区别：

- 贪婪的逻辑：
  - 我想要更多
  - -> 所以我选择一个大概率能赢的方式去争取
  - -> 哪怕冒险，也是计算过的冒险
- 27杂色梭哈的逻辑
  - 我现在很难受
  - -> 我要让这种难受立刻停止
  - -> 随便什么方式都行
  - -> 梭哈



27梭哈现象分析：为了逃离空仓痛苦而展开的情绪自救活动。

- 心态展现：哪怕拿到的是27杂色，照样有可能赢。没有那把剑我照样可以歼灭敌军！
- 客观事实：27杂色梭哈的人，心里清楚这牌大概率输；
- 行动目的：
  - 他不是冲着赢去的，他是冲着结束去的；
  - 或者说，他根本不在乎能不能赢，他只在乎能不能停止当下这种痛苦。
- 驱动分析：逃离某种痛苦。



进一步提出问题：为什么空仓会带来痛苦，使大部分人不得不重仓来缓解这种痛苦？



# 0x03 农耕文明的诅咒

### 3.1 生产资源的惯性误区

一句话：土地闲置即生存威胁。



现象：农业文明的人，对闲置有本能的不安——手停口停，人闲心慌。

逻辑：空仓=资金闲置=土地闲置=荒芜=饿死=生存威胁，这是刻进基因里的恐惧。

辅证：资金在银行躺着不会带来痛苦，一旦进入股市成为生产资料，散户就常常忍不住重仓。



展开描述：土地闲置即罪过——资源必须被持续利用

- 农耕特质：闲=没产出=生存威胁，操作=耕耘，必须做
  - 在农耕社会，土地是核心生产资料。让土地闲置（休耕除外）被视为浪费和失职，因为每一季的闲置都意味着潜在食物的损失；
  - 时间与土地绑定，无所作为直接等同于生存威胁。

- 海洋文明特质：
  - 等=保命=为下次准备
  - 操作=出海，看准了才做

- 股市映射
  - 资金即土地：投资者潜意识里将资金视为资本的土地，空仓就像让土地荒芜，会产生强烈的浪费感和错过感；
  - 行动伦理：总得种点什么的心态导致强行交易，即使机会不佳；
  - 关键区别：农耕中行动必有物理产出（除草、施肥总有痕迹），但股市中无效交易只会消耗手续费和精力。





### 3.2 生产逻辑的惯性误区

更深一层，为什么土地闲置=浪费？

- 因为土地耕种就会有收获，即一分耕耘一分收获；
- 农耕思想的误区
  - 误把重仓当作为所有土地撒上种子；
  - 误把频繁买卖当作辛勤耕耘；
  - 误把赚钱当做收获，而非了解市场、研究市场的认知提升。



农耕文明的劳动预期：有劳就会有得，劳而不得是小概率灾年。

海洋文明的劳动预期：劳而不得是常态，劳而有得是小概率的幸运。

股市：多动≠多赚，甚至多动=多错，在错误的天气里出海捕捞会导致丧命。





### 3.3 一句话概括诅咒

我宁愿做错，也不愿什么都不做。



### 3.4 我的个人体会

做事逻辑误区：妄想一分耕耘一分收获，把赚钱当做收获，而非眼界和知识的提升。



做事惯性：选择做一件事之前，我总会先根据已有的知识判断投产比，然后再判断可行性，最后再决定是否去做。

局限性：

- 很多事情是干中学，认识不断迭代，能看到更多的机会、更高的可行性；
- 妄想一分耕耘一分收获，本质上是一种自我设限，是一种确定性带来的诅咒。

个人经历：

- 有一段时间我在写量化交易程序，写出初版以后，我发现成交速度远远比不过别人，遂弃；

- 过段时间朋友写出来了成交速度前沿的程序，给了我信心，才继续写下去。



# 0x04 海洋文明决策模型

### 4.1 认知重构

什么是收获？

- 股市的耕耘不是频繁操作，收获不是赚钱；
- 股市的耕耘是研究市场，收获是对市场的认知在提升。





什么是闲置？

- 股市的闲置不是浪费，不是资源的沉睡与流逝；
- 股市的闲置是渔人拒绝在风暴中出海的智慧，是等待风浪平息、鱼群聚集的海域，并在此刻将船只与渔网打磨至锋利。



### 4.2 海洋文明：恶劣天气不出海

海洋捕捞的本质，并非持续不断的劳作，而是一套高度依赖自然周期、严格管理风险、并精准捕捉机遇的复杂生产系统。闲置在其中并非被动停止，而是系统运行的有机组成部分和主动战略选择。



海洋捕捞的生产周期可以清晰地分为几个阶段，其中闲置期占据了重要且积极的位置：

1. **观察与决策（岸上闲置期）**

   - 活动：老练的船长和渔民并不急于出海。他们研究潮汐、星象、历史渔汛记录（古代），或分析水文数据、卫星云图、探鱼报告（现代）。他们修补渔网、保养船只、补充给养；

   - 认知：这不是不生产，而是 生产的关键预备期 。信息的收集与研判，直接决定了后续行动的成败。此时的时间投入和资源保养，是生产效率的乘数。

2. **航行与定位（海上巡航闲置期）**
   - 活动：船只驶向目标海域。在此期间，船员持续观察海面鸟群、水色，使用探鱼仪扫描。渔网并未撒下；
   - 认知：这并非无效移动，而是 生产的必要位移和最终侦察 。航线的选择、航速的控制，都是为了以最佳状态抵达战场。此时的燃油消耗和人力投入，是捕获的必要成本。

3. **等待与忍耐（海上待机闲置期）**
   - 活动：抵达预判的渔场后，可能仍需等待鱼群聚集或到达最佳水层。船只可能随波漂流，引擎低速运转，全员保持警戒；
   - 认知：这是 生产机会的孵化期。过早撒网会惊散鱼群，过晚则错过时机。极致的耐心本身就是一种生产力，它确保资源在最高效的时刻被使用。



小结：恶劣天气不出海——最高级的战略闲置。





### 4.3 海洋文明格言

我宁愿什么都不做，也不愿做错。



### 4.4 海洋文明决策模型

研究天时、地利、人和。

- 在天时、地利、人和等条件好的时候出海。

- 在天时、地利、人和等条件差的时候少出海、不出海
  - 休息身体和精神
  - 研究什么时候天气好
  - 研究哪里的海域鱼获多
  - 补船补网磨鱼叉



出海之后尽人事，听天命。





### 4.5 使用轻仓替代空仓

兵法有正奇之变：就像兵法里的斥候小队一样，常年使用1成仓试探和观察市场，嗅到时机后再一击必杀。

海洋文明应该也有，去浅海试探和观察天气和鱼获。
