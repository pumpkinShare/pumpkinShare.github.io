# 0x01 股民分类

- 菜鸟冲动操作频繁交易——**渴求刺激的赌博，玩法问题**
- 第二等级：总在股票下跌时买入，并且根据他人的经验操作——**建仓问题**
- 第三等级：开始研究市场行情和趋势，形成自己的牛熊判断，独立决策，并且耐心坚守而不是试图抓住所有波动导致丢掉仓位——**独立决策问题、持仓问题**
- 第四等级：在判断正确的基础上，以合适的时机入场——**时机问题**





# 0x02 错误复盘

### 2025年

- 万华化学，
  - 估值修复的反转判断正确，但是浮盈加仓位置不对、反转的买入时机尚早、受朋友做T影响、受其他股票亏损影响**盘中**冲动全部清仓等，导致丢掉仓位。
  - 判断正确，**持仓错误**。
- 中国核建，
  - 7个点止盈半仓，剩余半仓没有到止损价，**盘中**不想忍受波动，导致丢掉仓位。
  - 判断正确，害怕盈利回撤，**持仓错误**。

- 白银有色，
  - 保证金上调带来的短期黑天鹅冲击恐慌，导致丢掉仓位。
  - **判断不准**，不计入错误。



### 2026年

- 军工ETF，
  - 判断两会利好，年前要买入，但是下跌反而**盘中**清仓去追涨其他股票，然后上涨再**盘中**买回来。
  - 年后第二天上涨有限，**盘中**冲动调仓至涨的多的股票，导致丢掉仓位。
  - 判断正确，因为涨得少而冲动追涨其他股票，**持仓错误**。

- 航天电子，
  - 听从他人建议买入，并且下跌加仓试图摊平成本，导致亏损。
  - 听从他人，**独立错误**。

- 生益科技，
  - 判断关键点有支撑，买入后恰逢百家放量，判断放量第三天、第四天是清仓时机。
  - 放量第二天涨得少、以及盈利达到预期，**盘中**卖出，导致丢掉仓位。
  - **盘中**卖出当天追涨其他股票，又在次日割掉错过第三天的启动。
  - 判断正确，**持仓错误**。

- 证券
  - 清仓股票的当天，判断证券下跌到位，**盘中**左侧买入全部交易仓位。
  - 临时性买入，计划错误，**仓位错误**。



### 小结

最高频：盘中冲动错误。

- 盘中冲动卖出盈利仓位。
- 盘中冲动追涨。



高频：仓位错误。



其他：独立错误。判断错误。



**行为模式：常常判断正确但结果亏损。**



# 0x03 改进措施

1、关闭实盘交易，只在盘后进行挂单交易，从源头上阻断冲动。

2、建立操作错误清单，操作前检查，每月定期回顾。

3、明确建仓比例和止损问题

- 以损定仓，每笔交易不希望损失超过交易仓位的2%
- 按照最少正常波动5%计算，任何情况下单笔建仓比例不能超过40%
